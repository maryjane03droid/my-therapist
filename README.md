# 🧠 My Therapist – Mental Wellness Web App
## Presenter name:Maryjane Wakuthii

A modern mental wellness web application built with React, Firebase, and AI integration.  
The app provides users with tools to track mood, reflect through journaling, access support groups, and chat with an AI assistant for emotional support.

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/maryjane03droid/my-therapist.git
cd my-therapist-app
npm install
VITE_GEMINI_API_KEY=your_api_key_here
npm run dev
npm run build
Deployment (Vercel)
Push your code to GitHub
Connect repo to Vercel
Add environment variables
Deploy
